Perguntas Frequentes

Posso rodar o instalador mais de uma vez?
Sim — é o comportamento esperado. Todo módulo verifica se já está instalado e atualizado antes de agir. Rodar de novo nunca duplica trabalho.
Preciso rodar como root?
Não. Rode como usuário normal — o sudo é solicitado apenas quando necessário, com uma sessão mantida viva durante toda a instalação.
O que acontece se um módulo falhar?
As ações daquele módulo específico são revertidas automaticamente (rollback em pilha). Módulos já instalados com sucesso não são afetados.
O instalador funciona sem interface gráfica (headless)?
Sim. Módulos que dependem de GUI (VS Code, Ghostty) detectam a ausência de sessão gráfica e são pulados automaticamente, sem interromper os demais.
Como desinstalo algo que instalei por engano?
Rode ./uninstall.sh. Ele lista os módulos registrados no manifesto local e permite escolher o que remover com segurança. Dados do usuário não são apagados.
